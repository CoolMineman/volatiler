# Volatiler

## Usage

Just Jitpack it for now

Make volatiler.txt in resources folder. Values seperated with tab
```
{intermediary class no net.minecraft. prefix (eg: class_1)}	{intermediary field no net.minecraft. prefix}	{full intermediary descriptor}	{Comment or whatever}
```
Example:
```
class_1297	field_18276	Lnet/minecraft/class_243;	Entity.velocity
```
